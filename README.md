# SZMC Geriatrics Pro: Universal v13.0

**The "Swiss Army Knife" for Geriatric Case Presentations.**

This tool allows you to drag & drop **any** hospital file (PDF Discharge Letter, PPTX Morning Report, Image of Handwritten Note) and instantly convert it into a structured Case Presentation or AI Prompt.

## New in v13.0
* **Crash-Proof Import:** Fixed PowerPoint (.pptx) parsing bugs. Now handles complex slide XML safely.
* **Auto-Save:** Never lose your work. Data is saved to your browser automatically as you type.
* **OCR:** Drag an image (JPG/PNG) of a document, and it will extract the text.
* **Smart Prompts:** If fields are empty, the AI prompt engine automatically grabs raw text to ensure you always get a result.

## How to Use
1.  Open `index.html` in Chrome/Edge on any hospital computer.
2.  **Drag & Drop** a file (e.g., a discharge summary PDF) into the sidebar.
3.  The tool extracts the text and attempts to fill the fields.
4.  Edit the data in the workspace.
5.  Click **"Generate AI Prompt"** to copy a professional prompt for ChatGPT/Claude.
6.  Click **"Download PPTX"** to get a ready-made slide deck for rounds.

## Privacy
* **Offline First:** All processing happens in your browser. No patient data is uploaded to any server.
* **Local Storage:** Data is saved only to your local device for convenience.
